# woodard-module-helpers

Shared helpers and CLI for Woodard Intelligence Platform modules.

See `docs/superpowers/specs/2026-04-20-phase-3-template-helpers-skills-design.md`
in the `woodard-energy/intelligence-platform` repo for the design spec.

## Install (in a module repo)

```toml
# pyproject.toml
dependencies = [
    "woodard-module-helpers @ git+https://github.com/woodard-energy/module-helpers@v0.1.0",
]
```

## CLI

```bash
woodard-cli --help
```
